# End-to-End Retail Performance Analytics

## 📌 Project Overview
Welcome to my retail analytics project! The goal of this project was to build a complete data pipeline to analyze North American retail performance. 

Starting with raw retail data, I utilized a full-stack data analysis approach: I cleaned and structured the dataset using **Python**, solved analytical problems and defined customer segments using **PostgreSQL**, and finally brought the insights to life through an interactive **Power BI** dashboard. 

The analysis covers a customer base of 3,900 individuals and $233K in total revenue, providing actionable insights into demographics, purchasing habits, and subscription growth opportunities.

## 🛠️ Tech Stack & Tools
* **Python:** Data cleaning, handling missing values, and structuring the raw dataset.
* **PostgreSQL:** Deep-dive exploratory data analysis (EDA), calculating loyalty segments, and querying complex business logic.
* **Power BI:** Data visualization, interactive dashboard creation, and final reporting.

## 📊 Dashboard Sections & Key Insights

The Power BI dashboard is broken down into five core analytical areas:

1. **Overview (Financials & Reach):** * Tracks high-level KPIs including $233K Total Revenue, $59.76 Average Order Value (AOV), and a 3.75/5 Average Rating.
   * Visualizes geographic distribution across North America and consistent seasonal sales.
2. **Segmentation (Demographics & Loyalty):** * Breaks down the customer base by gender (68% Male) and age groups.
   * Categorizes customers into loyalty tiers (Champion, Loyal, Regular, New) and Spend Tiers.
3. **Product (Items & Trends):** * Identifies top-performing categories (Clothing) and specific items (Blouses, Shirts).
   * Analyzes AOV by clothing size and highlights trending colors (Olive).
4. **Behavior (Purchasing Habits):** * Maps out purchase frequency (highest at "Every 3 Months").
   * Analyzes the impact of shipping methods (2-Day Shipping drives the highest AOV) and the usage of discounts vs. non-discounts.
5. **Subscription (Recurring Revenue):** * Highlights the current 27% subscriber rate.
   * Compares the lifetime value and satisfaction of subscribers vs. non-subscribers, identifying a massive opportunity to upsell current "Loyal" non-subscribers.
